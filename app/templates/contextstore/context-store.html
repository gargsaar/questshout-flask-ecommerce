Twilio Segment | Context Store (Concept)

Twilio Segment Context Store converts fragmented data coming from object cloud sources into an opinionated data model that seamlessly operates on the data coming from different sources. A data model is a set of rules that represent and validate the structure and format of data. At a high level, data models provide an abstract definition of a real-world object (such as a person) and outline what data should be included in each instance of that object (such as first name, last name, birthday, and so on). This data model that can then be used for improving the identity resolution by stitching together entities across touchpoints to create a holistic identity graph of profiles resolved across systems and building the “Golden Profile” that stitches together data from all sources (first-party data to data from 3P apps) to provide a holistic, accurate, and up-to-date view into user’s journey.

Context Store contains Objects and Relationships between Objects.
